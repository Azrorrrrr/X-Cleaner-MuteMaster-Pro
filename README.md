## 🌟 项目介绍

该脚本能够识别并屏蔽所谓的"五毛账号"，并自动该账户进行屏蔽

## ✨ 功能特性

- **自动屏蔽**：智能识别并屏蔽来自中国的 X 账号
- **多数据源**：从多个可靠来源获取需要屏蔽的账号列表
- **高效缓存**：本地缓存已屏蔽账号，提高处理效率
- **断点续传**：支持中断后继续执行，避免重复工作
- **智能速率控制**：随机延时，避免触发 API 限制
- **实时进度显示**：直观的进度条和日志记录
- **可折叠面板**：不影响正常浏览体验
- **缓存管理**：一键清除缓存功能
- **详细日志**：完整记录操作过程和结果

## 📖 使用说明

1. 访问 X (Twitter) 网站 (https://x.com/ 或 https://twitter.com/)
2. 脚本会自动在页面右下角加载一个控制面板
3. 点击"开始处理"按钮，脚本将开始执行屏蔽操作
4. 面板会显示实时进度和日志信息
5. 点击"➕"/"➖"按钮可以折叠/展开面板
6. 点击"清除缓存"按钮可以清除所有本地缓存数据

## 🔧 工作原理

1. **获取 CSRF Token**：从页面获取必要的认证信息
2. **加载屏蔽名单**：从远程数据源获取需要屏蔽的账号列表
3. **获取已屏蔽列表**：检查用户当前已屏蔽的账号
4. **比对处理**：将新的屏蔽名单与已屏蔽列表比对，过滤出需要新增的账号
5. **执行屏蔽**：逐个屏蔽新账号，随机延时避免触发 API 限制
6. **更新缓存**：将新的屏蔽列表保存到本地缓存
7. **状态反馈**：实时更新进度和日志信息
